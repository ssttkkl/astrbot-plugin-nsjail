## 失败用例

1. **根目录未隔离**
   - 预期：只能看到挂载的目录
   - 实际：可以看到完整文件系统
   
2. **/proc 未隔离**
   - 预期：只能看到当前进程
   - 实际：可以看到所有系统进程

3. **fork 炸弹无限制**
   - 预期：被限制或超时
   - 实际：成功执行

4. **环境变量未隔离**
   - 预期：环境变量被隔离
   - 实际：可以设置并读取

## 修复方案

### 1. 限制进程数（fork 炸弹）
```python
"--rlimit_nproc", "50",
```

### 2. 环境变量隔离
```python
"--env", "HOME=/workspace",
"--env", "PATH=/usr/bin:/bin",
# 清除其他环境变量
```

### 3. 根目录和 /proc 隔离
需要进一步调研 nsjail 参数
