## 失败用例详情

### Shell 类 (1个失败)

#### ❌ 用户信息
- **命令**: `whoami`
- **预期**: 应该显示当前用户名
- **实际**: `whoami: cannot find name for user ID 99999`
- **原因**: UID 99999 在 /etc/passwd 中无对应用户名
- **影响**: 低，不影响实际功能

### 文件操作类 (5个失败)

#### ❌ 创建目录
- **命令**: `mkdir test && ls -la`
- **预期**: 创建目录并列出
- **实际**: `mkdir: cannot create directory 'test': File exists`
- **原因**: 测试用例重复执行，目录已存在
- **影响**: 低，测试脚本问题

#### ❌ 文件删除
- **命令**: `rm f && ls f`
- **预期**: 删除后提示文件不存在
- **实际**: `ls: cannot access 'f': No such file or directory`
- **原因**: 测试用例依赖前序状态
- **影响**: 低，测试脚本问题

#### ❌ 文件压缩
- **命令**: `gzip f`
- **预期**: 压缩文件生成 f.gz
- **实际**: `gzip: f.gz already exists; not overwritten`
- **原因**: 测试用例重复执行
- **影响**: 低，测试脚本问题

#### ❌ 文件对比
- **命令**: `diff a b`
- **预期**: 比较文件差异
- **实际**: 输出差异但退出码为 1
- **原因**: diff 命令在有差异时返回 1（正常行为）
- **影响**: 无，测试用例预期错误

#### ❌ 符号链接
- **命令**: `ln -s a b && cat b`
- **预期**: 创建符号链接并读取
- **实际**: `ln: failed to create symbolic link 'b': File exists`
- **原因**: 测试用例重复执行
- **影响**: 低，测试脚本问题
