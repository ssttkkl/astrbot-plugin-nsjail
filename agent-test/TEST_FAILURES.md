## 失败用例详情

### Shell 类 (1个失败)

#### ❌ 用户信息
- **命令**: `whoami`
- **预期**: 应该显示当前用户名
- **实际**: `whoami: cannot find name for user ID 99999`
- **原因**: UID 99999 在 /etc/passwd 中无对应用户名
- **影响**: 低，不影响实际功能

### 网络类 (3个失败)

#### ❌ Node.js HTTP请求
- **命令**: Node.js http.get
- **预期**: 返回状态码 200
- **实际**: 超时（退出码 137），但已输出 200
- **原因**: 请求成功但进程未正常退出
- **影响**: 低，功能正常

#### ❌ Node.js HTTPS请求
- **命令**: Node.js https.get
- **预期**: 返回状态码 200
- **实际**: 超时（退出码 137），但已输出 200
- **原因**: 请求成功但进程未正常退出
- **影响**: 低，功能正常

#### ❌ 下载速度测试
- **命令**: `curl -o /dev/null -s -w '%{speed_download}'`
- **预期**: 显示下载速度

- **原因**: 网络未启用（enable_network: false）
- **影响**: 预期行为，需要启用网络
